# FlowMind

FlowMind is a modern, premium Kanban board built with React, TypeScript, and TailwindCSS. It features an Apple-inspired minimalist aesthetic, fluid drag-and-drop functionality, and a focus on crystal-clear user experience.

## ✨ Features

- **Apple-Inspired Design**: Minimalist, clean, and highly readable interface.
- **Fluid Drag-and-Drop**: Powered by `@dnd-kit` for a natural and responsive feel.
- **Time Tracking**: Automatically calculates time spent on tasks once moved to "Done".
- **Priority Management**: Clear visual badges for High, Medium, and Low priorities.
- **Modern Stack**: Built with React 19, TypeScript, Vite, and Zustand for state management.
- **Micro-interactions**: Smooth animations with Framer Motion.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/flowmind.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠️ Tech Stack

- **Frontend**: React 19, TypeScript
- **Styling**: TailwindCSS 4.0
- **State Management**: Zustand
- **Drag & Drop**: dnd-kit
- **Animations**: Framer Motion
- **Icons**: Lucide React
